# Kris Consultoria Financeira - Site de Jornada de Desbloqueio

Site one-page para a consultora financeira **Kris**, com foco em orientar clientes sobre o **Crédito do Trabalhador**.

## Visão geral

O site foi projetado como uma experiência de progressão e descoberta:

- **Conheça** a Kris
- **Entenda** o Crédito do Trabalhador
- **Contrate** com confiança

A página utiliza um design moderno, responsivo e móvel-first, com animações de scroll, indicador de progresso e uma celebração visual ao chegar na etapa de contratação.

## O que está incluído

- `index.html`: página principal com toda a estrutura, estilos e lógica JavaScript.
- `AGENT_CONTEXT.md`: documento de contexto para ajudar agentes a entender rapidamente o propósito, a estrutura e as interações do site.

## Pontos importantes

- A experiência é narrativa, sem bloqueios artificiais: o usuário pode rolar livremente entre fases.
- Há um CTA direto para contratação e um CTA alternativo para contato via WhatsApp.
- A animação de confetes respeita `prefers-reduced-motion` e executa apenas uma vez.
- O botão WhatsApp está configurado com um número fictício e deve ser atualizado antes do lançamento.
